GhostNet: A Python-Based Stock Reminder for Cryptocurrencies and Investments

GhostNet is a simple yet powerful Python application designed to track investments and returns for various cryptocurrencies or assets. Whether you’re a cryptocurrency enthusiast or an investor in other assets, GhostNet allows you to manage your financial data efficiently by providing an easy-to-use interface to record investments, track returns, and display overall statistics.

This project is open-source and available on GitHub.

Features of GhostNet

  • Investment and Return Management: Users can add, edit, or delete investment and return data for any currency or asset.
  • Data Persistence: The application saves all data to a file (data.dat), ensuring that your data remains intact even after the program exits.
  • Statistics Display: GhostNet calculates and displays total investments, returns, and profit/loss statistics.
  • Simple and Intuitive Interface: With a text-based interface, the program is user-friendly, even for those who are new to programming.

How It Works

GhostNet operates by reading and writing data to a file, and it uses a simple command-line interface (CLI) for interaction. Here is an overview of the main functionality of the code:

  1. Reading Data: The read_data function loads the investment and return data from a file. If the file does not exist, it creates a new one with a header line for currency, investment, and return values.

  2. Writing Data: The write_data function saves the current state of the data to the file, ensuring that updates such as new investments or changes in return values are preserved.

  3. Adding Investment and Return: Users can add investment and return amounts for any currency using the add_investment and add_return functions. These functions automatically update the data and display a message with the new values.

  4. Editing Data: The edit_data function allows users to modify the investment and return values for a specific currency.

  5. Deleting Data: The delete_data function deletes the data for a specific currency, ensuring that outdated or incorrect information can be removed.

  6. Displaying Statistics: The display_statistics function provides an overview of total investments, returns, and profit/loss for all currencies. It also shows a breakdown of each currency’s investment and return.

  7. Formatting: The format_currency function ensures that all currency values are displayed in a readable format with commas and a specified currency symbol.

Code Overview

File Structure

The project consists of the following main components:

  • ghostnet.py: The main Python file that runs the application.
  • data.dat: The file where investment and return data is stored.

Code Breakdown

  1. Reading and Writing Data: The functions read_data and write_data handle reading from and writing to the data.dat file. The file is formatted with a header that labels each column (currency, investment, return).

  2. Adding, Editing, and Deleting Data: Functions like add_investment, add_return, edit_data, and delete_data manipulate the data stored in memory. The changes are then written back to the data.dat file to persist the updates.

  3. Displaying Statistics: The display_statistics function calculates the total investment, total return, and the overall profit or loss from all the currencies, helping you track your financial performance.

  4. Currency Formatting: The format_currency function formats numbers in a user-friendly way, adding commas for thousands and appending a currency symbol.
